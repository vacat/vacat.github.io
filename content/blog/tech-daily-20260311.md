### 英伟达×UIUC：400万 token 上下文新 SOTA

英伟达与 UIUC 华人团队提出高效训练方法，将 LLM 上下文从 128K 扩展至惊人的 **400万 token SOTA 纪录**！

**技术亮点：**
- 基于 Llama3.1-Instruct 打造 UltraLong-8B 模型
- 特殊文档分隔符 + YaRN 位置编码扩展技术
- 单步持续预训练策略比多步方法更高效
- 256 个 NVIDIA H100 GPU 训练，1M/2M/4M 模型分别仅需 5/6/13 小时

该模型在长上下文基准测试（RULER、LV-Eval、InfiniteBench）中表现卓越，同时在标准任务上保持顶尖竞争力。

## 🧩 Agent 框架与 MCP 协议

### 微软发布 MCP C# SDK 1.0

微软宣布官方 MCP（Model Context Protocol）C# SDK 正式达到 **1.0 版本**，完整支援 2025年11月25日版 MCP 规格。

**重点更新：**
- 授权探索与权限管理（渐进式授权同意机制）
- 工具与资源的中继资料扩充
- 敏感资讯处理流程（URL 模式互动引导）
- 长時間 HTTP 请求处理机制
- 任务（Tasks）功能支持

这一更新让 .NET 开发者可通过成熟 C# 工具链构建符合最新 MCP 规格的客户端与服务器。

> 🔗 [来源：iThome](https://www.ithome.com.tw/news/174287)

### OpenAI、Microsoft 相继宣布支持 MCP

继 MCP 规范更新后，**OpenAI 宣布将支持 MCP**  across 其产品线，Agents SDK 已可用，ChatGPT 桌面版和 Responses API 即将支持。Sam Altman 表示这是连接 AI 与数据生态的重要一步。

微软同步宣布支持 MCP，并推出 **Playwright-MCP server**，允许 Claude 等 AI Agent 通过 Chrome 无障碍树浏览网页和与站点交互。
